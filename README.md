## 📂 Proyectos Desarrollados

### 💻 Sistema de Control de Inventario Tecnológico Policial
*Desarrollado en Entorno de Prácticas Profesionales — Comisaría*

*   **Descripción:** Solución digital orientada a la administración y fiscalización rigurosa de los recursos tecnológicos de la dependencia. Reemplaza los formatos físicos de control por un flujo automatizado que monitoriza el estado operativo y la asignación del equipamiento.
*   **Componentes Clave:**
    *   Trazabilidad completa de entradas y salidas de hardware esencial: laptops, PCs de escritorio, radios de comunicación y cámaras de seguridad.
    *   Módulo de asignación de responsabilidades con bitácora de auditoría histórica e inmutable.
    *   Alertas automatizadas sobre el estado técnico de los equipos y reportes listivos para inspecciones internas.
*   **Tecnologías:** PHP, JavaScript, SQL Server / MySQL, CSS Estructurado.
*   **Resultados:** Transición exitosa hacia un entorno cero papel, reduciendo discrepancias en el inventario de hardware a un 0% y acelerando los tiempos de entrega de equipos.

### 💧 Sistema Inteligente de Gestión de Servicios Básicos
*Plataforma Integral para la Red de Comisarías*

*   **Descripción:** Aplicación diseñada para centralizar la supervisión, consumo y estado operativo de los servicios públicos críticos (agua potable, energía eléctrica y conectividad a internet) en la totalidad de las sedes policiales de la jurisdicción.
*   **Componentes Clave:**
    *   Dashboard analítico para el monitoreo de facturaciones, consumos históricos y alertas por picos anómalos.
    *   Módulo de reportes en tiempo real para registrar fallas de conectividad o cortes en el suministro eléctrico.
    *   Base de datos optimizada para el almacenamiento cronológico y consolidación de datos financieros de cada sede.
*   **Tecnologías:** Node.js, React, MySQL, Bootstrap, HTML5.
*   **Resultados:** Proporcionó una visión clara y unificada de los costos operativos, reduciendo tiempos de gestión administrativa y previniendo interrupciones en los servicios de las comisarías.
